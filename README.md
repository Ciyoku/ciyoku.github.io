# المكتبة الأخبارية

موقع مكتبة عربية ثابت (Static Site) لعرض الكتب، البحث داخل النصوص، وتصفح المحتوى عبر قارئ مخصص.

## المزايا
- فهرس كتب مع بحث سريع.
- صفحة بحث في نصوص الكتب.
- صفحات المؤلفين والتصنيفات.
- قارئ كتب يدعم:
  - الفهرس (الفصول)
  - الأجزاء
  - التنقل بين الصفحات
  - حفظ موضع القراءة
  - المفضلة

## التشغيل محليًا
شغّل المشروع عبر أي خادم Static محلي (لا يُنصح بفتح الملفات مباشرة عبر `file://`).

أمثلة:
- VS Code Live Server
- `python -m http.server 8080`

ثم افتح:
- `http://localhost:8080/index.html`

## التحقق قبل النشر
يتضمن المشروع سكربتات تحقق للهيكل والبيانات:

```bash
npm run check
```

أوامر مفصلة:
- `npm run check:js`
- `npm run check:books`
- `npm run check:shell`
